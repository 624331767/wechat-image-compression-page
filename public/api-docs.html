POST

        
/api-dns/dns-records

        
添加DNS记录

        

            

                zoneId - 域名区域ID（必填）
            

            

                type - 记录类型（必填）
            

            

                name - 记录名称（必填）
            

            

                content - 记录内容（必填）
            

            

                ttl - TTL值（必填）
            

            

                proxied - 是否启用代理（必填，布尔值）
